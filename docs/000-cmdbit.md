cmdbit.exe
----------

Определяет битность запускаемых файлов в формате [WIN](https://habr.com/ru/post/266831/) или [ELF](https://habr.com/ru/post/480642/)  
Синтаксис:

```
cmdbit.exe path/to/program.exe
```

- вернет `32`, если `program.exe` 32х битный.
- вернет `64`, если `program.exe` 64х битный.
- вернет `unknown`, во всех остальных случаях.

Кейсы использования:
--------------------

- Если в качестве проверяемого файла указать обычный текстовый файл,  
  то поскольку он и не `32` и не `64`, то результат будет `unknown`.  

- Если запустить утилиту, не указав путь к проверяемому файлу, 
  тогда в stderr будет выдено: `invalid syntaxis.`  

- Если запустить утилиту, указав не существующий файл, 
  например, файл `no_exist`, тогда в stderr будет выдено: 
  `main(std::exception): can not open: 'no_exist'`  

- Если файл существует, но по каким то причинам его не удалось открыть,
  (например, не хватило прав), тогда в stderr будет выдено: 
  `main(std::exception): описание-неисправности`
